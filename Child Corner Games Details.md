# Child Corner Games Details

This document contains the detailed design and gameplay mechanics for two children’s games: The Little Painter and Math Adventure. These games are planned as part of an educational and entertainment corner for children.

## 2\. Math Adventure Game

Math Adventure is an educational game designed to make children enjoy mathematics through interaction and simple challenges.

### A. Main Menu

The game starts with an attractive interface containing four large and clear buttons representing the mathematical operations:

* **Addition (+)**
* **Subtraction (-)**
* **Multiplication (×)**
* **Division (÷)**

### B. Challenge and Question System

* After selecting an operation, a math question appears. (Example: 5 + 3 = ?)
* Multiple-choice answers are displayed below the question.
* **Attempt System:** The child has 3 attempts (hearts) per round.

  * Wrong Answer Behavior, One attempt is deducted. An encouraging message appears asking the child to try again.
  * End of Round, When all attempts are used, the round ends and the final result is displayed.

### C. Achievement and Reward System

* **Instant Feedback:** When the child answers correctly, Animations appear (stars, balloons, etc.), Encouraging sound effects are played
* **Achievement Badge:** If the child completes a certain number of correct answers without losing attempts, they receive A badge Or a trophy, These rewards are added to the child’s in-game profile.
